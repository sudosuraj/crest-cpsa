Appendix J: Databases

CREST CPSA Certification Practice Questions

Learn database security assessment for the CREST CPSA certification. This appendix covers the security of major database platforms including Microsoft SQL Server and Oracle, as well as the security of database connectivity from web and application tiers.

Start Practicing Appendix J Questions

Topics Covered

  • J1 Microsoft SQL Server - SQL Server security, authentication modes, xp_cmdshell, and privilege escalation
  • J2 Oracle RDBMS - Oracle security, TNS listener, default accounts, and PL/SQL injection
  • J3 Web/App/Database Connectivity - Connection strings, ODBC/JDBC security, and database access from applications

Database security is critical as databases often contain the most sensitive information in an organization. Understanding how to assess database security, identify misconfigurations, and exploit common vulnerabilities is essential for any penetration tester.
